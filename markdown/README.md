# Tekstien lähdeaineistot

Tästä hakemistosta löytyvät RYTV aineistoissa käytettävät lähdeaineistot markdown muodossa. Näistä dokumenteista koostetaan varsinaiset RYTV aineistot.

## Kirjoitustyön tueksi

RYTV Osa A -projektissa on luotu joitain markdown tiedostoja aihepiireittäin, jotka sisältävät viittauksia mm. YTV2012 aineistoihin. Näitä aineistoja on mahdollista käyttää lähtötietona RYTV sisältöjen luomisessa. Alla viittaukset näihin tiedostoihin.
